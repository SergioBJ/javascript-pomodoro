# Pomodoro App

Aplicación Pomodoro desarrollada con JavaScript Vanilla.

## Características

- Crear tareas
- Iniciar temporizador Pomodoro
- Marcar tareas completadas automáticamente
- Temporizador de descanso
- Interfaz responsive

## Tecnologías

- HTML5
- CSS3
- JavaScript
